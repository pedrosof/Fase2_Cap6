<p class=MsoNormal><b><span style='font-family:"Apple Color Emoji";mso-bidi-font-family:
"Apple Color Emoji"'>&#128220;</span> Descrio<o:p></o:p></b></p>

<p class=MsoNormal><i>O projeto tem como base o problema de uma colheita de
cana de acar mais eficiente. A colheita manual  mais eficiente pois faz um
corte mais rente ao cho, no entanto ela  mais danosa ao trabalhador e por
isso o uso de colheitadeiras mecnicas vem sendo mais utilizado.<o:p></o:p></i></p>

<p class=MsoNormal><i><o:p>&nbsp;</o:p></i></p>

<p class=MsoNormal><i>O simples uso da colheita mecnica, causa uma perda de
15% de matria prima em relao a colheita manual. Por isso, tcnicas de
otimizao de anlise de colheita precisam ser utilizadas para reduzir esta
perda.<o:p></o:p></i></p>

<p class=MsoNormal><i><o:p>&nbsp;</o:p></i></p>

<p class=MsoNormal><i>Este script analisa imagens, sensores de brix e clima
para saber o melhor momento para colheita e ter melhor aproveitamento do
produto.<o:p></o:p></i></p>
